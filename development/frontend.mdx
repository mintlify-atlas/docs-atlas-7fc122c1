The Fork's frontend is a React single-page application built with Create React App, Tailwind CSS, and modern React patterns.

## Tech stack

- **React 19**: UI framework with concurrent features
- **React Router 7.5**: Client-side routing
- **Tailwind CSS 3.4**: Utility-first CSS framework
- **Axios**: Promise-based HTTP client
- **Craco**: CRA configuration override
- **Playwright**: End-to-end testing framework

### IntensityToggle

Three-way toggle for selecting conversation intensity: mild, savage, or brutal.

<Accordion title="IntensityToggle component code">

```jsx
import React from "react";
import { INTENSITY } from "../utils/constants";

export const IntensityToggle = ({ value, onChange, disabled }) => {
  return (
    <div
      data-testid="intensity-toggle"
      className="grid grid-cols-3 gap-2 rounded-2xl border border-white/10 bg-black/30 p-2"
    >
      {Object.entries(INTENSITY).map(([key, meta]) => {

        return (
          <button
            key={key}
            type="button"
            disabled={disabled}
            data-testid={`intensity-toggle-${key}`}
            onClick={() => onChange(key)}
            className={`rounded-xl px-3 py-2 ${
              active
                ? "bg-crimson/20 border border-crimson/30"
                : "bg-white/5 hover:bg-white/10"
            }`}
          >
            <div className="flex items-center justify-between">
              <span className="font-medium text-zinc-100">{meta.label}</span>
              {active ? (
                <span className="h-2 w-2 rounded-full bg-crimson" />
              ) : (
                <span className="h-2 w-2 rounded-full bg-white/10" />
              )}
            </div>
            <div className="mt-1 text-xs text-zinc-400">{meta.hint}</div>
          </button>
        );
      })}
    </div>
  );
};
```

### ChatWindow

The main chat interface that handles message display and composition.

<Accordion title="ChatWindow component code (excerpt)">

```jsx
import React, { useState, useEffect, useRef } from "react";
import axios from "axios";
import { MessageBubble } from "./MessageBubble";
import { INTENSITY, uuidv4 } from "../utils/constants";

const API = `${process.env.REACT_APP_BACKEND_URL}/api`;

export const ChatWindow = ({ forkStatement, intensity, sessionId }) => {
  const [messages, setMessages] = useState([]);
  const [draft, setDraft] = useState("");
  const [loading, setLoading] = useState(false);
  const [error, setError] = useState("");

  const listRef = useRef(null);

  const send = async () => {
    const text = draft.trim();
    if (!text || loading) return;

    setError("");
    const userMsg = {
      id: uuidv4(),
      role: "user",
      content: text,
      createdAt: new Date().toISOString(),
    };

    setMessages((prev) => [...prev, userMsg]);
    setDraft("");
    setLoading(true);

    try {
      const res = await axios.post(`${API}/chat`, {
        forkStatement,
        intensity,
        sessionId,
        messages: [...serverMessages, { role: "user", content: text }],
      });

      const reply = res?.data?.reply;
      if (!reply) throw new Error("Empty reply");

      const alterMsg = {
        id: uuidv4(),
        role: "alter",
        content: reply,
        createdAt: new Date().toISOString(),
      };

      setMessages((prev) => [...prev, alterMsg]);
    } catch (e) {
      setError("The other door jammed. Try again.");
    } finally {
      setLoading(false);
    }
  };

  const onKeyDown = (e) => {
    if (e.key === "Enter" && !e.shiftKey) {
      e.preventDefault();
      send();
    }
  };

  return (
    <section data-testid="chat-window">
      {/* Message list */}
      <div ref={listRef} className="h-[460px] overflow-y-auto">
        {messages.map((m) => (
          <MessageBubble
            key={m.id}
            side={m.role === "user" ? "right" : "left"}
            label={m.role === "user" ? "You" : "Other You"}
            content={m.content}
          />
        ))}
      </div>

      {/* Composer */}
      <textarea
        data-testid="composer-input"
        value={draft}
        onChange={(e) => setDraft(e.target.value)}
        onKeyDown={onKeyDown}
        rows={3}
      />
      <button
        data-testid="composer-send-button"
        onClick={send}
        disabled={loading || !draft.trim()}
      >
        Say It
      </button>
    </section>
  );
};
```

</Accordion>

**Key features**:
- Auto-scrolls to bottom when new messages arrive
- Enter to send, Shift+Enter for newline
- Loading state with typing indicator
- Error handling with user-friendly messages
- Optimistic UI updates (adds user message immediately)

## Custom hooks

### useSessionId

Manages session ID persistence in localStorage.

<Accordion title="useSessionId hook code">

```javascript
import { useMemo } from "react";
import { uuidv4 } from "../utils/constants";

export const useSessionId = () => {
  const sessionId = useMemo(() => {
    const existing = window.localStorage.getItem("fork.sessionId");
    if (existing) return existing;
    const id = uuidv4();
    window.localStorage.setItem("fork.sessionId", id);
    return id;
  }, []);

  return sessionId;
};

export const useSessionIdReset = () => {
  return () => {
    const id = uuidv4();
    window.localStorage.setItem("fork.sessionId", id);
    return id;
  };
};
```

</Accordion>

**Usage**:

```jsx
const sessionId = useSessionId();
const resetSession = useSessionIdReset();

const handleReset = () => {
  const newSessionId = resetSession();
  setMessages([]);
  setStarted(false);
};
```

## State management

The Fork uses **local component state** with React hooks:

```jsx
function Home() {
  const [forkStatement, setForkStatement] = useState("");
  const [intensity, setIntensity] = useState("mild");
  const [started, setStarted] = useState(false);
  const [showReset, setShowReset] = useState(false);
  
  const sessionId = useSessionId();
  
  // Props drilling to child components
  return (
    <>
      <ForkSetup
        forkStatement={forkStatement}
        setForkStatement={setForkStatement}
        intensity={intensity}
        setIntensity={setIntensity}
        onStart={() => setStarted(true)}
      />
      {started && (
        <ChatWindow
          forkStatement={forkStatement}
          intensity={intensity}
          sessionId={sessionId}
        />
      )}
    </>
  );
}
```

**Why no global state?**
- Single page application with simple data flow
- All state is ephemeral (no persistence beyond localStorage)
- Component hierarchy is shallow
- Props drilling is minimal and manageable
