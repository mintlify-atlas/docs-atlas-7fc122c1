  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get The Fork running locally in minutes with Docker Compose or native setup.
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the repository and navigate into the project directory.
      
      ```bash
      git clone https://github.com/CassieMarie0728/the-fork.git
      cd the-fork
      ```
    </Step>

    <Step title="Configure environment variables">
      Copy the environment templates and add your Emergent LLM API key.
      
      ```bash
      cp backend/.env.example backend/.env
      cp frontend/.env.example frontend/.env
      # Edit backend/.env and set EMERGENT_LLM_KEY
      ```

      <Note>The `EMERGENT_LLM_KEY` is required for chat functionality. Without it, the chat endpoint will return a 500 error.</Note>
    </Step>

    <Step title="Start with Docker Compose">
      Launch all services (MongoDB, backend, frontend) with a single command.
      
      ```bash
      docker-compose up --build
      ```

      The app will be available at:
      - Frontend: http://localhost:3000
      - Backend API: http://localhost:8000
      - API docs: http://localhost:8000/api/docs
    </Step>

    <Step title="Try your first conversation">
      Open the app, enter a fork statement (e.g., "I chose engineering instead of art"), select an intensity level, and start the conversation with Other You.
    </Step>
  </Steps>
